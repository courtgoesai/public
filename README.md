# Skills by Courtney

Free Claude skills you can install and use. Each one lives in its own folder under `skills/`.

New here? A skill is a small set of instructions you give your AI. Think of it like a job description for a specific task. Once installed, you trigger it with a phrase and Claude knows what to do.

## Available skills

| Skill | What it does | Folder |
|-------|--------------|--------|
| **LLM Council** | Turns one question into five expert opinions, plus a final verdict. For decisions where being wrong is expensive. | [skills/llm-council](skills/llm-council) |
